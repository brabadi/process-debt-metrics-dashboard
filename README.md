Modelo de métricas para la evaluación de la deuda de procesos

Este repositorio contiene los instrumentos desarrollados para la aplicación del modelo de métricas propuesto 
para la evaluación de la deuda de procesos en proyectos de desarrollo de software.

Contenido:
Dashboard en Excel: herramienta automatizada para el cálculo y visualización de resultados.
Dashboard en PDF: versión manual para registro y presentación de resultados.

Instrucciones de uso:
Abra el archivo de dashboard en Excel.
Ingrese los valores obtenidos al aplicar cada una de las métricas del modelo.
En caso de que una métrica no aplique, registre el valor como N/A según corresponda.

El instrumento calculará automáticamente:
Promedio de causas
Promedio de impactos
Nivel global de deuda de procesos
Visualice los resultados en los indicadores gráficos dinámicos, los cuales representan el nivel de deuda según la escala definida.

Interpretación de resultados:
Los valores obtenidos se interpretan de acuerdo con la escala definida en el modelo:

 0% ≤ valor ≤ 20% nivel bajo
 
21% ≤ valor ≤ 50% nivel moderado

51% ≤ valor ≤ 80% nivel alto 
    
valor > 80% nivel crítico

Consideraciones:
No es obligatorio aplicar todas las métricas; el modelo es adaptable según el contexto del proyecto.
Los resultados permiten identificar la deuda de procesos, no definir estrategias de mitigación.
El instrumento está diseñado para facilitar la interpretación y análisis de los resultados.

Acceso:
Este repositorio puede ser utilizado como material de apoyo para la aplicación del modelo en contextos académicos o profesionales.
